# RentalCar — Приложение для аренды автомобилей 🚗

Современное веб-приложение для поиска и бронирования автомобилей. Проект выполнен с акцентом на высокую производительность, адаптивность и удобство пользователя.

## 🚀 Стек технологий

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** CSS Modules (Mobile First подход)
- **Icons:** React SVG Components / Lucide-react
- **Deployment:** [Vercel](https://vercel.com/)

## ✨ Основные функции

- **Каталог авто:** Удобный просмотр доступных моделей с фильтрацией.
- **Страница деталей:** Полная информация об автомобиле (характеристики, условия аренды, аксессуары).
- **Форма бронирования:** Интерактивная форма для мгновенной заявки.
- **Адаптивный дизайн:** Идеальное отображение на десктопах, планшетах и мобильных устройствах.
- **Оптимизация:** Использование `next/image` для быстрой загрузки изображений и корректных LCP показателей.
